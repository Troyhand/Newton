# Contributing to Newton

Newton is currently in early-stage development.

Public contribution guidelines will evolve as the project matures.

## Current Contribution Focus

At this stage, contributions should focus on:

- documentation clarity
- architecture discussion
- terminology refinement
- validation workflow concepts
- Personal Assistant development concepts
- safe domain pack structure
- user interface ideas

## Contribution Boundaries

Newton is not intended to become an unrestricted knowledge-editing platform.

Future contributions related to Domain Packs, governance logic, validation workflows, or Session Share will require structured review, provenance checks, and clear approval standards.

## Protected Components

Some Newton components may remain private, protected, or source-available under separate terms.

These may include:

- governed substrate internals
- proprietary agent-language mechanisms
- signed validation infrastructure
- stewardship systems
- high-trust governance logic

## Direction

Newton welcomes careful builders, researchers, and contributors interested in drift-resistant, auditable, Personal Assistant-first cognitive infrastructure.
